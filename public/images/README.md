# Imagens do site

Coloque aqui as imagens e atualize as URLs em `src/content/siteCopy.ts`:

- **Hero:** imagem principal (sugestão: 800×600 px ou maior)
- **O que fazemos:** desenvolvimento/software (720×480 px)
- **Tecnologias:** stack/ambiente tech (720×480 px)
- **Sobre:** equipe ou trabalho (720×480 px)

Exemplo em `siteCopy.ts`:

```ts
image: "/images/hero.jpg"
```

Atualmente o site usa imagens de placeholder (Unsplash). Substitua pelos caminhos acima quando tiver as fotos.
